variable:

	the value change during execution of the program.

variable naming rules:
	*) must start with alphabets (a-z)
		or _ (underscore)
	*) do not start with numbers (0-9)
	 but we can start second letter onwards
	*) keywords or reserve words not allowed 
	ex: if for else while class .......
	*) special chars not allowed in variablename except _
	ex:~!@#$%^&*()-=+|<>?:"{}[];',./ 
	*) space is not allowed in variable name.
	*) variable name must be unique.
